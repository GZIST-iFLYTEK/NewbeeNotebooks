# 视频要求

1. 时长适配：完整版 7 分钟（适配院校交流、行业展会），可剪辑 3 分钟精简版（保留核心成果、重点采访、标识亮点），适配社交媒体传播。

2. 色调与音乐：校园场景暖色调为主，企业、实验室冷色调点缀；全程使用轻快激昂的纯音乐，采访段落音乐减弱，旁白、成果展示段落音乐适度增强。

3. 采访要求：采访对象提前熟悉台词，表情自然，镜头以中近景为主，背景贴合身份（领导配成果手册、教师配课堂场景、学生配实践场景）。
